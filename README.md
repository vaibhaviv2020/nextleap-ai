# 🚀 NextLeap AI – Skill Gap Analyzer

NextLeap AI is an AI-powered career guidance assistant that helps users identify missing skills and discover the right path toward their target career goals.

Built using **Google ADK** and powered by **Gemini**, the agent analyzes a user’s current skills and generates structured, personalized recommendations instantly.

---

# 🎯 Problem Statement

Many students and professionals face confusion while planning their careers because:

* ❌ They don’t know which skills they are missing
* ❌ There is no clear roadmap toward a target role
* ❌ Online resources are overwhelming and unstructured
* ❌ Generic career advice lacks personalization

NextLeap AI simplifies this process using AI-driven skill gap analysis and actionable guidance.

---

# ✨ Features

* 🧠 AI-powered skill and career goal analysis
* 📊 Skill gap identification using Gemini reasoning
* 🎯 Personalized recommendations and next steps
* ⚡ Fast and structured response generation
* ☁️ Deployable on Google Cloud Run
* 🔥 Lightweight single-agent ADK architecture

---
